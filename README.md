# Depee



## Installation ｜ 安装

环境检测:

* [NodeJs](#a-node-js).
* [Gulp](#b-gulp-js).
* [Bower](#c-bower).

### a. 安装nodejs

安装nodejs，并设置环境变量

```console
brew install nodejs
```


### 安装bower

安装:

```console
# npm install bower
```


### 安装gulp

```console
$ npm install gulp
```

## 运行

### 运行开发服务器:

```console
$ gulp serve
```

### 打包:

```console
$ gulp build
```
